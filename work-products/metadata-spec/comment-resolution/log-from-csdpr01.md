# Comment Resolution Log

The table summarizes the comments that were received during the 30-day public review of the committee specification draft "[Data Provenance Metadata Version 1.0](https://docs.oasis-open.org/dps/prov-meta/v1.0/prov-meta-v1.0.html)" and their resolution.
Comments came to the technical-committee-comments and the dps-comment mailing lists in five messages providing a total of 72 comments.
[The public review did start May 13, 2026 and did end June 12, 2026 at 23:59 UTC](https://groups.oasis-open.org/discussion/invitation-to-comment-on-dps-tcs-data-provenance-metadata-version-10-csd01).

A status of "Completed" in the Disposition column indicates that the editors have implemented the changes on which the TC decided, which are outlined in the Resolution column. It also is a hyperlink to the GitHub commit notice.

To process these 72 comments in a traceable but reasonable way, the editors created five container issues on GitHub and linked from there to the 72 individual comments for further processing.

Two labels were conceived to ease the access to the 72 leave issues [pm10-csdpr01-fdbck-leaf](https://github.com/oasis-tcs/dps/issues?q=is%3Aissue%20state%3Aopen%20label%3Apm10-csdpr01-fdbck-leaf) and the five container issues [pm10-csdpr01-fdbck](https://github.com/oasis-tcs/dps/issues?q=is%3Aissue%20state%3Aopen%20label%3Apm10-csdpr01-fdbck).
